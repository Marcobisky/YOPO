# Digital Image Processing Lab1

## 环境设置

创建并激活conda环境：
```bash
conda env create -f environment.yml
conda activate image-processing-lab1
```

## 运行

```bash
cd code
python main.py
```

## 说明

变量规范:

- 类名: 开头大写驼峰
- 类中 utility 性质的方法 (函数): 下划线开头
- 标量、向量、一般函数: 开头小写下划线分割
- 常量: 全大写下划线
- 类实例变量: 每个单词开头大写下划线
